# Event Commerce SaaS

Multi-tenant SaaS platform for event venues.

## Stack

- Java 21
- Spring Boot 3
- Spring Data JDBC
- PostgreSQL
- Flyway
- Angular
- GitHub Actions

## Modules

- Tenant
- Venue
- EventSpace
- Catalog
- Quotation
- Payment
- Tax
- Supplier

## Run locally

mvn spring-boot:run
